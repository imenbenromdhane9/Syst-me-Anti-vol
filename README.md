# Syst-me-Anti-vol
Description : Système complet combinant suivi GPS et détection de mouvements suspects pour véhicule.

Fonctionnalités :

-Géolocalisation GPS

-Détection de mouvements via capteur de vibration / accéléromètre

-Application mobile Android : visualisation GPS, alertes en temps réel, activation/désactivation du système

Technologies : Android, Firebase Cloud, GPS, capteurs de mouvement

Rôle : Développement mobile et IoT
